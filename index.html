# Player Age RAT 365 Watermark Salary Trade Value Surplus

Methodology

Trade values are distributed from a fixed salary pool equal to the sum of every current NBA salary. No money is created or removed ‚Äî the system only redistributes existing salary based on performance.

The Watermark rating blends three components to smooth out single-season volatility and injury impact: A) Average RAT per game across all available seasons (up to 5 years), B) Average RAT per game over the last 3 seasons, and C) Average RAT per game in the current 2025-26 season. Components are only included if the player has data for that period. Hover over any Watermark value to see the breakdown.

The final Trade Value is a 75/25 blend: 75% from the Watermark-based distribution and 25% from the RAT 365-based distribution. Both distributions use an exponent of 1.4, creating a progressive curve where elite production is rewarded exponentially.

An age factor of ¬±20% is then applied. The league average age serves as the neutral point ‚Äî younger players receive up to a 20% bonus, older players up to a 20% penalty, scaling linearly. After the age adjustment, all values are renormalized so the total still equals the salary pool. Hover over any age to see the exact multiplier.

Step 1  Watermark = avg( A: all-seasons RAT/g, B: last-3yr RAT/g, C: 2025-26 RAT/g )
Step 2  TVWM = (max(Watermark,0)^1.4 √∑ Œ£ all WM scores) √ó Pool
Step 3  TVRAT = (max(RAT365,0)^1.4 √∑ Œ£ all RAT scores) √ó Pool
Step 4  TVblend = 75% √ó TVWM + 25% √ó TVRAT
Step 5  Age Mult = 1 + 0.20 √ó (avg_age ‚àí player_age) √∑ max_deviation
Step 6  Trade Value = normalize( TVblend √ó Age Mult ) to Pool
